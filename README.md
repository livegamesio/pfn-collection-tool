# PFN Collection Tool

Generate random numbers 


## Installation


```bash
git clone https://github.com/livegamesio/pfn-collection-tool.git
```

```bash
cd pfn-collection-tool
```

```bash
yarn install
```

```bash
chmod +x app.js
```

## Usage
```bash
yarn start --help
or
./app.js --help
```

## Generate
```bash
yarn start --count 100
```
